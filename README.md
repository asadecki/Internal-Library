Internal-Library
================

1. Install MongoDB
    `brew update`
    `brew install mongodb`
2. Start MongoDB
    a. Install brew services
    `brew tap homebrew/boneyard`
    b. Start service
    `brew services start mongodb`
    c. Run MongoDB
    `mongo`
3. Install from NPM
    `npm install`
4. Initial grunt
    `grunt`
5. Run Internal-Library
    `grunt serve`
